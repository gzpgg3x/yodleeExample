2. To run Yodlee NodeJS Sample App
2.1. Upzip the NodeJS Sample file
2.2. Open command prompt console:
- Go to Yodlee NodeJS Sample App folder
- Run command to update libraries: npm install 
- Run command to start: node .\bin\www 
2.3. Open link http://localhost:3000/ in your browser
